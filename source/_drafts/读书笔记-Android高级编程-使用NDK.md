## 第2章 深入了解Android NDK

Android NDK的主要内容

+ ARM、x86、MIPS交叉编译器
+ 构建系统
+ Java原生接口头文件
+ C 库
+ Math 库
+ POSIX 线程
+ 最小的C++库
+ ZLib 压缩裤
+ 动态链接库
+ Android 日志库
+ Android 像素缓冲区库
+ Android 原生应用APIs
+ OpenGL ES 3D 图形库
+ OpenSL ES 原生音频库
+ OpenMAX AL 最小支持
